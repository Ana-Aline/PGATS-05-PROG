1. Abra a pasta do projeto
2. Clique Terminal
3. Clique em New Terminal
4. Digite: npm init -y
5. Pressione ENTER
6. Crie uma pasta “src”
7. Crie uma pasta “test”
8. Abra o arquivo package.json
9. Altere o valor da propriedade “type” e “commonjs” para “module”
10. Instale o package “mocha” no seu projeto digitando npm i mocha no Terminal
